# Cub3d

## Installation

### Clone the repository

```shell
git clone git@github.com:Sleleu/Cub3d.git
cd Cub3d
```

### Install the minilibx library

In the ./Cub3d folder :

```shell
git clone https://github.com/42Paris/minilibx-linux.git mlx_linux
make
```
